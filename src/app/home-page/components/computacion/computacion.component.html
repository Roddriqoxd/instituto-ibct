Contenido
Windows 10 Aprenderás a encender, apagar y navegar por una computadora de manera segura. Te enseñaremos a organizar carpetas, mover archivos, instalar programas básicos y personalizar tu entorno de trabajo en el sistema operativo Windows.
Microsoft Word Aprende a crear documentos profesionales como cartas, informes y currículums. Usarás herramientas de formato, insertarás imágenes, tablas y trabajarás el diseño del contenido con claridad y estilo.
Microsoft Excel Aprende a trabajar con hojas de cálculo, fórmulas básicas, porcentajes, promedios y gráficos. Ideal para tareas escolares, presupuestos personales o trabajos de oficina.
Microsoft PowerPoint Aprende a crear presentaciones visuales usando texto, imágenes, animaciones y transiciones. Ideal para exposiciones escolares, clases y reuniones de trabajo.
Internet Aprenderás a navegar por la web, buscar información confiable, manejar un navegador, y utilizar herramientas como Google, YouTube, Zoom y redes sociales.
Microsoft Access Aprende a crear bases de datos, tablas, formularios y consultas. Ideal para organizar información y automatizar tareas administrativas.
WinRAR Aprende a comprimir archivos en formato .RAR o .ZIP para ahorrar espacio y facilitar el envío. También aprenderás a extraer archivos y protegerlos con contraseña.
WinZip Descubre cómo usar WinZip para comprimir documentos, organizarlos, enviarlos por correo y protegerlos con cifrado si es necesario.
Microsoft Publisher Aprende a crear folletos, tarjetas, invitaciones y otros materiales impresos. Este curso es ideal para quienes quieren iniciarse en el diseño gráfico básico.
Dactilografía Mejora tu velocidad y precisión al escribir. Aprenderás la ubicación correcta de los dedos, ejercicios prácticos y técnicas para escribir sin mirar el teclado.
