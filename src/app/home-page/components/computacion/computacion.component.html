Contenido
Manejo del Sistema Operativo (Windows) Aprenderás a encender, apagar y navegar por una computadora de manera segura. Te enseñaremos a organizar carpetas, mover archivos, instalar programas básicos y personalizar tu entorno de trabajo en el sistema operativo Windows. Este conocimiento es la base para usar cualquier equipo con confianza.
Microsoft Word – Redacción y Formato de Documentos Domina la creación de documentos profesionales: cartas, informes, currículums y más. Aprenderás a dar formato al texto (negritas, márgenes, fuentes), insertar tablas, imágenes y organizar la información de manera clara. Ideal para estudiantes, trabajadores y emprendedores.
Microsoft Excel – Cálculos y Organización de Datos Te enseñaremos a usar hojas de cálculo para realizar operaciones básicas (sumas, promedios, porcentajes), manejar datos en tablas, y crear gráficos visuales. Este contenido es clave para tareas administrativas, presupuestos personales o trabajo de oficina.
Internet, Correo Electrónico y Herramientas Web Aprenderás a navegar por internet de forma segura, a buscar información útil, a crear y manejar una cuenta de correo electrónico, enviar y recibir archivos, y usar plataformas digitales como Google, YouTube y Zoom. Estas habilidades son esenciales hoy en día para estudiar, trabajar o comunicarte.
